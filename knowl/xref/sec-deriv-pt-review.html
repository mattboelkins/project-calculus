Exploration 1.3.1.

In Section 1.1, we learned how the average velocity of a moving object, such as a car driving on a straight road, is determined by the quantity
\begin{equation*} \frac{\text{change in position}}{\text{change in time}}\text{.} \end{equation*}
More generally, the average rate of change of a function \(f\) from \(x = a\) to \(x = b\) is given by
\begin{equation*} AV_{[a,b]} = \frac{f(b)-f(a)}{b-a} = \frac{\text{change in } f}{\text{change in } x}\text{.} \end{equation*}
For example, if \(f(x) = 4x - x^2\text{,}\)
\begin{equation*} AV_{[2,4]} = \frac{f(4) - f(2)}{4-2} = \frac{0 - 4}{2} = -2 \end{equation*}
The average rate of change of a function is important because it provides information about how a function is changing.
